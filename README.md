# 简易文件处理程序（Huan-File-Tool）

这是一个使用JavaScript编写的建议文件处理程序，使用`ES Module`模式。

目前，所有导出函数位于：`src/main.cjs`和`src/main.mjs`文件中，此文件也是本项目的入口文件。

注意：本项目同时支持`CommonJS`和`ESModule`两种模式。
